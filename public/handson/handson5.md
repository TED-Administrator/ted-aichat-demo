# AIとWeb検索

## はじめに

前のページで、AIエージェントの特徴のひとつに「**ツール利用**（検索などの道具を使うこと）」があると学びました。このページでは、それを実際に体験します。

ここでいう「Web検索」とは、AIが**インターネットで調べもの**をすることです。AIは、勉強し終わった日（**カットオフ**）までの知識しか持っていません。でも、あなたが普段やっているように「キーワードで検索して、ページを開いて読む」という操作をAI自身にやらせれば、最新の情報にもたどり着けます。

> 💡 Web検索が使えるのは、この「**AIとWeb検索**」ページを開いているときだけです。ほかのページ（1〜4ページ目）では、AIは検索せず、自分の知識だけで答えます。

---

## 第1章：AIの知識には限界がある

### 仕組み

AIは「勉強が終わった日」より後に起きたことを知りません。また、毎日変わる情報（天気・値段・ニュースなど）も、勉強した中には入っていません。

```
【知識だけのAI】
質問 →（覚えた知識だけで答える）→ 古い・あいまいな答えになることがある

【道具を使えるAI】
質問 → 🔍 ネットで調べる → 📄 ページを読む → 最新の情報で答える
```

つまり、人間が「知らないことはネットで調べる」のと同じことを、AIにもさせてあげればよいのです。

---

## 第2章：人間のWeb検索を思い出してみよう

### 仕組み

あなたは知らないことを調べるとき、無意識に次の3ステップを踏んでいます。

```
① 検索ボックスにキーワードを入れる   （例：「山の日 2026」）
        ↓
② 出てきた一覧から、よさそうなものを選ぶ
        ↓
③ そのページを開いて、本文を読む
```

AIエージェントも、まったく同じことを2つの道具（ツール）でやります。

| 人間の操作 | AIの道具 | はたらき |
|-----------|-----------|---------|
| キーワードで検索する | **web_search** | キーワードで検索し、結果の一覧（タイトル・URL）を受け取る |
| ページを開いて読む | **open_url** | 選んだURLを開いて、本文を読む |

---

## 第3章：AIに検索させてみよう

### やってみよう①：最新の情報を調べてもらう

通常モード（推論OFF）で、次のプロンプトを送ってみましょう。

```
2026年の山の日は何月何日ですか？最新の情報を調べて教えてください。
```

送信すると、答えの上に **「🔍 Webで調べています...」** というカードが出てきます。これを開くと、AIが今やっている操作が見えます。

- 🔍 **○○を検索しました** … web_search でキーワード検索した
- 📄 **○○を読みました** … open_url でページを開いて本文を読んだ

AIは「検索 → 結果から選ぶ → ページを開く」を自分でくり返し、最後に分かったことをまとめて答えます。まさに、あなたが普段やっている調べものを、AIが代わりにやっているのです。

> 💡 うまく見つからないと、AIはキーワードを変えて何度も検索し直すことがあります。人間と同じ「試行錯誤」をしている様子に注目してみましょう。

### やってみよう②：日々変わる情報を聞いてみる

```
明日の東京の天気を調べて教えてください。
```

> 💡 天気やニュースのように「今この瞬間」の情報は、AIの知識だけでは絶対に答えられません。検索の道具があって、初めて答えられる質問です。

---

## 第4章：URLを直接渡して読ませる

### 仕組み

検索をはさまずに、「このページを読んで」と**URLを直接わたす**こともできます。これも、人間が「このリンク先を要約して」と頼むのと同じです。

### やってみよう③：指定したページを要約させる

```
https://ja.wikipedia.org/wiki/紫式部 のページを開いて、どんな人物か2行で教えてください。
```

> 💡 AIは open_url でそのページの本文を取りこみ、内容を読んでから要約します。答えの最後に、参照したページのURLが示されることも確認しましょう。

---

## 第5章：AIが調べた情報も鵜呑みにしない

### 仕組み

ネット上の情報には、**古い情報・間違った情報・かたよった情報**も混ざっています（3ページ目で学んだ「プロンプト・インジェクション」のように、AIをだまそうとするページもあり得ます）。

AIが検索して答えてくれても、それがいつも正しいとは限りません。

### やってみよう④：出どころを確かめる

AIが答えの中で示したURLを、**自分のブラウザでも開いて**みましょう。AIの要約と、実際のページの中身が合っているかを確かめます。

> 💡 大事な調べものでは、AI任せにせず「いくつかのページで確かめる」「公式サイトなど、いちばん元の情報にあたる」習慣が大切です。これは、人間がネット検索するときの心得とまったく同じです。

---

## まとめ

| 学んだこと | ポイント |
|--------|---------|
| 知識の限界 | AIは勉強した時点までの知識しかなく、最新情報や日々変わる情報は知らない |
| 人間と同じ手順 | ①キーワード検索 → ②結果を選ぶ → ③ページを開いて読む |
| 2つの道具 | web_search（検索）と open_url（ページを開く）でAIが自分で調べる |
| エージェントの実感 | あなたが普段やっているWeb検索を、AIが代わりにやってくれる |
| 情報の確認 | AIが調べた情報も鵜呑みにせず、出どころや元の情報を自分でも確かめる |

「考えるだけ」だったAIが、道具を使って「自分で調べて動く」ようになりました。これがAIエージェントの第一歩です。普段あなたがやっている操作を思い浮かべながら、いろいろな質問で試してみてください。
