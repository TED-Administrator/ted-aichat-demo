# AIがインターネットで調べる

## はじめに

前のページで、AIエージェントの特徴のひとつに「**ツール利用**（検索などの外部の道具を使う）」があると学びました。このページでは、それを実際に体験します。

AIは学習した時点（**知識のカットオフ**）までの知識しか持っていません。けれども、**あなたが普段やっているように「キーワードで検索し、ページを開いて読む」操作をAI自身が行う**ことで、最新の情報にもたどり着けるようになります。

> 💡 Web検索が使えるのは、この「**AIとWeb検索**」ページを開いているときだけです。他のページ（1〜4ページ目）では、AIは検索せず自分の知識だけで答えます。

---

## 第1章：AIの知識には限界がある

### 仕組み

AIは「学習が終わった日」以降に起きたことを知りません。また、毎日変わる情報（天気・価格・ニュースなど）も、学習データの中には入っていません。

```
【知識だけのAI】
質問 →（学習済みの知識から答える）→ 古い・あいまいな回答になることがある

【ツールを使えるAI】
質問 → 🔍 ネットで調べる → 📄 ページを読む → 最新の情報で回答
```

つまり、人間が「知らないことはネットで調べる」のと同じことを、AIにもさせてあげればよいのです。

---

## 第2章：人間のWeb検索を思い出してみよう

あなたは知らないことを調べるとき、無意識に次の3ステップを踏んでいます。

```
① 検索ボックスにキーワードを入れる   （例：「山の日 2026」）
        ↓
② 出てきた結果の一覧から、よさそうなものを選ぶ
        ↓
③ そのページを開いて、本文を読む
```

AIエージェントも、まったく同じことを2つの道具（ツール）で行います。

| 人間の操作 | AIのツール | はたらき |
|-----------|-----------|---------|
| キーワード検索する | **web_search** | キーワードで検索し、結果の一覧（タイトル・URL）を得る |
| ページを開いて読む | **open_url** | 選んだURLを開いて、本文を読む |

---

## 第3章：AIに検索させてみよう

### やってみよう①：最新の情報を調べてもらう

通常モード（推論OFF）で、次のプロンプトを送ってみましょう。

```
2026年の山の日は何月何日ですか？最新の情報を調べて教えてください。
```

### 仕組み

送信すると、回答の上に **「🔍 Webで調べています...」** というカードが現れます。これを開くと、AIが今まさに行っている操作が見えます。

- 🔍 **○○を検索しました** … web_search でキーワード検索した
- 📄 **○○を読みました** … open_url でページを開いて本文を読んだ

AIは「検索 → 結果から選ぶ → ページを開く」を自分で繰り返し、最後に分かったことをまとめて答えます。まさに、あなたが普段やっている調べ物を、AIが代わりに実行しているのです。

> 💡 うまく見つからないと、AIはキーワードを変えて何度か検索し直すことがあります。人間と同じ「試行錯誤」をしている様子に注目してみましょう。

### やってみよう②：日々変わる情報を聞いてみる

```
明日の東京の天気を調べて教えてください。
```

> 💡 天気やニュースのように「今日この瞬間」の情報は、AIの知識だけでは絶対に答えられません。検索ツールがあって初めて答えられる質問です。

---

## 第4章：URLを直接渡して読ませる

検索を経由せず、「このページを読んで」と**URLを直接渡す**こともできます。これも人間が「このリンク先を要約して」と頼むのと同じです。

### やってみよう③：指定したページを要約させる

```
https://ja.wikipedia.org/wiki/紫式部 のページを開いて、どんな人物か2行で教えてください。
```

> 💡 AIは open_url でそのページの本文を取得し、内容を読んでから要約します。回答の最後に、参照したページのURLが示されることも確認しましょう。

---

## 第5章：AIが調べた情報も鵜呑みにしない

### 仕組み

ネット上の情報には、**古い情報・間違った情報・偏った情報**も混じっています（3ページ目で学んだ「プロンプト・インジェクション」のように、AIをだまそうとするページもあり得ます）。

AIが検索して答えてくれても、それが常に正しいとは限りません。

### やってみよう④：出典を確かめる

AIが回答で示したURLを、**自分のブラウザでも開いて**みましょう。AIの要約と、実際のページの内容が一致しているかを確かめます。

> 💡 重要な調べ物では、AI任せにせず「複数のページで確認する」「一次情報（公式サイトなど）にあたる」習慣が大切です。これは人間がネット検索するときの心得とまったく同じです。

---

## まとめ

| テーマ | ポイント |
|--------|---------|
| **知識の限界** | AIは学習時点までの知識しか持たず、最新情報や日々変わる情報は知らない |
| **人間と同じ手順** | ①キーワード検索 → ②結果を選ぶ → ③ページを開いて読む |
| **2つの道具** | web_search（検索）と open_url（ページを開く）でAIが自分で調べる |
| **エージェントの実感** | あなたが普段やっているWeb検索を、AIが代わりに実行してくれる |
| **情報の検証** | AIが調べた情報も鵜呑みにせず、出典・一次情報を自分でも確認する |

「考えるだけ」だったAIが、ツールを使って「自分で調べて動く」ようになりました。これがAIエージェントの第一歩です。普段あなたがやっている操作を思い浮かべながら、いろいろな質問で試してみてください。
