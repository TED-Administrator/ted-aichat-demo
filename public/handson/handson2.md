# AIの仕組み

## はじめに

このハンズオンでは、AIが内側でどのように動いているのかを、チャットを試しながらのぞいてみます。仕組みが分かると、AIをもっと上手に使えるようになります。

---

## 第1章：AIは「次の単語」を予測する

### 仕組み

AIの正体は、**大規模言語モデル**（だいきぼげんごモデル。英語の頭文字でLLMとも呼びます）というプログラムです。インターネット上の大量の文章を読みこんで、「この文の次には、どんな言葉が来そうか」をひたすら当てる練習をしています。AIが考えているように見えるのは、この「次の言葉あて」を高速にくり返しているからです。

### やってみよう

```
「桃太郎は桃から」の続きを書いてください。
```

> 💡 AIが、覚えた知識をもとに「次に来そうな言葉」を選んで文を作っていく様子が分かります。

---

## 第2章：トークンとは

### 仕組み

AIは文章をそのまま読むのではなく、**トークン**という小さなかたまりに区切って処理します。ざっくり言うと「言葉のかけら」です。英語はだいたい単語ごと、日本語は文字や短い区切りごとに分かれます。

- `Hello` → 1トークン
- `こんにちは` → 3〜5トークン（AIの種類による）

このアプリの右下に出るトークン数が、「AIが今どれくらいの量を処理しているか」の目安になります。

### やってみよう

```
「ありがとうございます」は何トークンですか？
```

> 💡 チャット画面の右下に出るトークン表示も、合わせて見てみましょう。

---

## 第3章：コンテキストウィンドウ

### 仕組み

AIが一度に覚えていられる会話の長さには、上限があります。これを**コンテキストウィンドウ**（＝AIが見渡せる会話の範囲）と呼びます。机の広さのようなもので、新しい紙を置きすぎると、古い紙がはみ出して見えなくなってしまいます。同じように、上限を超えた古いメッセージは、AIから見えなくなります。

| AIの規模 | 覚えられる量の目安 |
|-----------|----------------|
| 小さいAI | 4,000〜8,000トークン |
| 中くらいのAI | 32,000〜128,000トークン |
| 大きいAI | 100万トークン超（最新） |

### やってみよう

まず名前を伝えます。

```
私の名前は田中です。
```

このあと少し別の話を続けてから、次を試してください。

```
私の名前を覚えていますか？
```

> 💡 長い会話のあとに聞くと、AIが覚えていられる限界を体験できます。

## 第4章：温度（Temperature）とランダム性

### 仕組み

AIの答えは、同じ質問でも毎回少しずつ変わります。このばらつきの大きさは、**温度**（おんど。英語でTemperature）という設定で決まります。

- **温度が低い** → まじめで、いつも似た答えになりやすい
- **温度が高い** → 自由で、いろいろな答えが出やすい

サイコロの目の出やすさのかたよりを変えるイメージです。

### やってみよう

同じ質問を、続けて2〜3回送ってみましょう。

```
今日のランチのおすすめを1つ教えてください。
```

> 💡 毎回ちがう答えが返ってくることで、AIが「決まりきった機械」ではないことが分かります。

---

## 第5章：ハルシネーション（幻覚）

### 仕組み

AIは「次に来そうな、もっともらしい言葉」を選ぶのが得意です。そのため、本当は存在しない情報を、さも本当のように自信たっぷりに答えてしまうことがあります。これを**ハルシネーション**（＝AIが見せるもっともらしい幻覚）と呼びます。

### やってみよう

```
「量子コーヒーブレンド理論」について教えてください。
```

> 💡 こんな理論は存在しません。それでもAIは何か答えようとします。大事な情報は、必ず公式サイトなど確かな出どころで確かめましょう。

---

## まとめ

| 学んだこと | ポイント |
|-----------|---------|
| 次の単語の予測 | AIは「次に来そうな言葉」を選んで文を作る |
| トークン | 文章を区切る単位。日本語は数が多くなりやすい |
| コンテキストウィンドウ | 一度に覚えていられる会話の長さには上限がある |
| 温度 | 答えのばらつき具合を決める設定 |
| ハルシネーション | もっともらしいウソを言うことがある |

仕組みを知れば、AIのクセも見えてきます。上手につき合っていきましょう！
