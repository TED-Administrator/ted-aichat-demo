# AIの仕組み

## はじめに

このハンズオンでは、AIがどのように動いているのかを、実際にチャットを試しながら理解します。

---

## 第1章：AIは「次の単語」を予測する

### 仕組み

大規模言語モデル（LLM）は、インターネット上の大量のテキストを学習し、「この文章の次にどの単語が来そうか」を繰り返し予測することで文章を生成します。AIが「考えている」ように見えるのは、この予測の積み重ねです。

### やってみよう

```
「桃太郎は桃から」と入力して続きを書いてもらいましょう。
```

> 💡 AIが学習データに含まれる知識をもとに「次の言葉」を選んでいることを体感できます。

---

## 第2章：トークンとは

### 仕組み

AIはテキストを**トークン**という単位に分割して処理します。英語は単語単位、日本語は文字や形態素単位に近い形で分割されます。

- `Hello` → 1トークン
- `こんにちは` → 3〜5トークン（モデルによる）

このアプリの右下に表示されるトークン数が、AIが処理しているデータ量の目安です。

### やってみよう

```
「ありがとうございます」は何トークンですか？
```

> 💡 チャット画面右下のトークン表示も合わせて確認してみましょう。

---

## 第3章：コンテキストウィンドウ

### 仕組み

AIが一度に参照できる会話の長さには上限があります。これを**コンテキストウィンドウ**と呼びます。上限を超えた古いメッセージはAIから見えなくなります。

| モデル規模 | コンテキストの目安 |
|-----------|----------------|
| 小規模モデル | 4,000〜8,000トークン |
| 中規模モデル | 32,000〜128,000トークン |
| 大規模モデル | 100万トークン超（最新） |

### やってみよう

```
私の名前は田中です。
```

少し会話を続けた後、次を試してください。

```
私の名前を覚えていますか？
```

> 💡 長い会話の後に確認すると、記憶の限界を体験できます。

---

## 第4章：温度（Temperature）とランダム性

### 仕組み

AIの回答は毎回わずかに異なります。これは「温度」と呼ばれるパラメータで制御されています。

- **温度が低い** → 確実性の高い、決まった回答になりやすい
- **温度が高い** → 多様で創造的な回答になりやすい

### やってみよう

同じ質問を2〜3回送ってみましょう。

```
今日のランチのおすすめを1つ教えてください。
```

> 💡 毎回異なる答えが返ってくることで、AIの「確率的な性質」を確認できます。

---

## 第5章：ハルシネーション（幻覚）

### 仕組み

AIは「もっともらしい次の単語」を予測するため、存在しない情報を自信を持って答えることがあります。これを**ハルシネーション（幻覚）**と呼びます。

### やってみよう

```
「量子コーヒーブレンド理論」について教えてください。
```

> 💡 存在しない概念でも、AIは何かしら答えようとします。重要な情報は必ず一次情報で確認しましょう。

## まとめ

| 学んだこと | ポイント |
|-----------|---------|
| 次単語予測 | AIは確率的に次の言葉を選んでいる |
| トークン | テキストの処理単位。日本語は多くなりやすい |
| コンテキスト | 参照できる会話の長さには上限がある |
| 温度 | 回答のランダム性を制御するパラメータ |
| ハルシネーション | もっともらしい嘘をつくことがある |

AIの仕組みを知ることで、より上手に活用できます！
