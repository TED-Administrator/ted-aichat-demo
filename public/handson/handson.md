# AIリテラシー ハンズオン

## はじめに

このハンズオンでは、AIチャットを実際に操作しながら、AIの仕組みや特性を体験的に学びます。

---

## 第1章：AIに話しかけてみよう

### やってみよう

左のチャット欄に、以下のメッセージを送ってみましょう。

```
こんにちは！あなたはどんなことができますか？
```

### ポイント

- AIは**文章でやり取り**します
- 質問は自然な日本語で書いてかまいません
- 送信は `Enter`、改行は `Shift + Enter` です

---

## 第2章：AIの得意・不得意を知ろう

### 知識のカットオフとは

AIは学習したデータに基づいて回答します。そのため、**学習データの締め切り日（カットオフ）以降の出来事**は知りません。

### やってみよう

```
あなたの知識のカットオフはいつですか？
今日のニュースを教えてください。
```

> 💡 2つ目の質問でAIがどう答えるか注目してみましょう。

---

## 第3章：プロンプトの工夫

AIへの指示（**プロンプト**）の書き方を変えると、回答の質が変わります。

### 悪い例と良い例

| プロンプト | 問題点 |
|-----------|--------|
| `教えて` | 何を教えるか不明 |
| `Pythonについて教えて` | 範囲が広すぎる |
| `Python の `for` 文の基本構文と使い方を、初心者向けに例を使って説明してください` | ✅ 対象・目的・形式が明確 |

### やってみよう

以下のプロンプトを試してみましょう。

```
小学生にもわかるように、AIとは何かを3つの箇条書きで説明してください。
```

---

## 第4章：マークダウンと数式

AIはマークダウン記法で回答することがあります。このアプリはそれに対応しています。

### やってみよう

```
二次方程式 ax² + bx + c = 0 の解の公式を
$$記法のブロック数式で書いてください。
```

期待される表示：

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

| 学んだこと | ポイント |
|-----------|---------|
