# AIリテラシー

## はじめに

このハンズオンでは、AIチャットを実際に動かしながら、AIがどんなものかを体で覚えていきます。むずかしい知識は要りません。気軽に話しかけてみましょう。

---

## 第1章：AIに話しかけてみよう

### 仕組み

AIとのやり取りは、すべて「文章」で行います。スマホでメッセージを送るのと同じ感覚です。聞きたいことを、ふだんの日本語でそのまま書いて大丈夫です。

### やってみよう

左のチャット欄に、次のメッセージを送ってみましょう。

```
こんにちは！あなたはどんなことができますか？
```

> 💡 送信は `Enter` キー、文の途中で改行したいときは `Shift + Enter` です。

---

## 第2章：AIの得意・不得意を知ろう

### 仕組み

AIは、ある日までのインターネットの文章を勉強して作られています。この「勉強し終わった日」を**カットオフ**（＝知識のしめ切り日）と呼びます。だから、それより後に起きたことや、今日のニュースは知りません。

### やってみよう

```
あなたの知識のカットオフはいつですか？
今日のニュースを教えてください。
```

> 💡 2つ目の質問にAIがどう答えるか見てみましょう。「分かりません」と正直に言えるかどうかがポイントです。

---

## 第3章：プロンプトの工夫

### 仕組み

AIへの指示や質問のことを**プロンプト**（＝AIへのお願いの文）と呼びます。同じことを聞くのでも、お願いの仕方をくわしく書くほど、返ってくる答えがよくなります。

### 良い例・悪い例

| プロンプト | どうか |
|-----------|--------|
| `教えて` | ❌ 何を教えるのか分からない |
| `メールを書いて` | ❌ 誰に・何のためかが分からない |
| `取引先への納期遅れのお詫びメールを、件名・本文・署名の形で、ていねいな言葉で書いてください` | ✅ 相手・目的・形が決まっている |

### やってみよう

下のプロンプトを試してみましょう。具体的にお願いすると、答えもしっかりしてきます。

```
新入社員向けに、ビジネスメールの基本マナーを5つのポイントで説明してください。
```

> 💡 「誰に・何を・どんな形で」を入れるのがコツです。

---

## 第4章：マークダウンと数式

### 仕組み

AIは、見出しや表、数式などをきれいに整えた形で答えることがあります。この書き方を**マークダウン**と呼びます。このアプリは、その表示にちゃんと対応しています。

### やってみよう

```
二次方程式 ax² + bx + c = 0 の解の公式を
$$記法のブロック数式で書いてください。
```

このように表示されれば成功です。

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

> 💡 数式が、教科書のようにきれいに表示されることを確認しましょう。
