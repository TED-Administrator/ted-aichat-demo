# AIの推論とエージェント

## はじめに

最近のAIは、「答えを出す前に、いったん考える」力を持つようになりました。このページでは、AIの**推論**（すいろん＝じっくり考えること）がどう働くのか、そしてそれが、自分で動く「**AIエージェント**」という新しい進化にどうつながるのかを学びます。

---

## 第1章：AIの推論とは

### 仕組み

これまでのAIは、質問を受け取るとすぐに答えを返していました。でも**推論モード**では、最終的な答えを出す前に、頭の中で「考えるプロセス」をはさみます。

> **【ふつうのモード】**
> 質問 → 答え
>
> **【推論モード】**
> 質問 → 考える（推論） → 答え

考えるプロセスの中で、AIはこんなことをしています。

| 推論でやること | 例 |
|-----------|-----|
| **問題を分ける** | 「この問題は3つに分けて考えよう」 |
| **似た例をさがす** | 「これに似たケースを思い出すと…」 |
| **自分で直す** | 「あ、さっきの考えは間違いだ。こっちが正しい」 |
| **見くらべる** | 「A・B・Cのやり方があるが、今回はBがいちばんいい」 |

推論なしのAIが「直感」でパッと答えるとすれば、推論ありのAIは「じっくり考えて」から答えます。特に算数や論理パズル、ややこしい問題では、推論モードのほうが**正解率がぐっと上がります**。

---

## 第2章：推論の仕組みを体験しよう

### やってみよう①：推論をONにしてみよう

入力欄の左にある **💡 電球のアイコン** ボタンを押すと、推論モードが**ON**になります（アイコンがオレンジ色に光ります）。

推論をONにした状態で、次のプロンプトを順番に試してみましょう。

```
1から10までの整数の合計を求めてください。
```

回答が返ってきたら、続けて次のプロンプトを送ってみましょう。

```
この計算の手順を教えてください。
```

> 💡 答えの上に「思考プロセス」が表示されます。AIが答えを出す前に、どう考えているかをのぞいてみましょう。

### やってみよう②：推論で複雑な問いを整理してみよう

推論をONにしたまま、次のプロンプトを試してみましょう。

```
「テレワーク週3日」と「毎日出社」で迷っています。それぞれのメリット・デメリットを整理して、判断のポイントを教えてください。
```

> 💡 日常的な問いでも、推論モードではさまざまな角度を「考えながら」整理していきます。思考プロセスでどのように組み立てているかに注目してみましょう。

## 第3章：推論からAIエージェントへ

### 仕組み

推論する力が十分に育ったとき、AIは単なる「質問に答える機械」を超えました。それが **AIエージェント** です。「エージェント」は「代理人（人の代わりに動く人）」という意味です。

> **【チャットAI】（これまで）**
> 人間 → 質問 → AI → 答え → 人間
>
> **【AIエージェント】（これから）**
> 人間 → 目標を与える → AI が自分で「考え・動き・直す」をくり返す → 結果

AIエージェントには、こんな特徴があります。

| 特徴 | 内容 |
|------|------|
| **計画を立てる** | 目標に向かう手順を、自分で組み立てる |
| **道具を使う** | 検索・計算・プログラム実行などの道具を使う |
| **自分で直す** | やった結果を見て、計画を直しながら進める |
| **長い仕事をやりきる** | 何段階もある作業を、自分で最後までやりとげる |

たとえば「旅行の計画を立てて」と頼んだとき——

**AIチャットの場合：**
文章でプランを提案するだけ

**AIエージェントの場合：**

1. 飛行機と宿の空き状況を検索する
2. 値段や条件を見くらべる
3. カレンダーと照らし合わせて、いちばん良い日を選ぶ
4. 予約まで自動でやりとげる

つまり、**「考えるだけ」から「実際に動く」へ**の進化です。

### やってみよう③：エージェントのような考え方を体験

推論をONにして、いくつもの段階がある仕事を頼んでみましょう。

```
私は来月、初めて東京から京都へ日帰り旅行を計画しています。予算は交通費含めて2万円以内です。どのように計画を立てるべきか、検討すべき要素を整理しながら考えてください。
```

> 💡 AIが計画を立てるときの「考え方の組み立て」に注目してください。これがいろいろな道具と組み合わさると、エージェントとして自分で動けるようになります。

---

## まとめ

| 学んだこと | ポイント |
|-----------|---------|
| 推論 | 答える前に考えるプロセス。問題を分ける・自分で直す・見くらべる |
| 推論の効果 | ややこしい問題や、論理が必要な問いで特に役に立つ |
| AIエージェント | 推論＋道具＋自分で直す＝自分で目標を達成できるAI |
| これからの姿 | AIは「聞かれたら答える道具」から「目標を与えると動くパートナー」へ進化している |

AIの推論する力は、ただ「賢くなる」だけでなく、AIのあり方そのものを変えつつあります。この進化を知ることで、AIをより上手に、そして安全に使えるようになります。
