# AIの推論とAIエージェント

## はじめに

最近のAIは「答えを出す前に考える」機能を持つようになりました。このページでは、AIの**推論**がどのように機能するのか、そしてそれがどのようにして自律的に行動する「**AIエージェント**」という新しい進化へとつながっているかを学びます。

---

## 第1章：AIの推論とは

### 仕組み

これまでのAIは、質問を受け取ると即座に回答を生成していました。しかし**推論モード**では、最終的な回答を出す前に「思考プロセス」を踏みます。

```
【通常モード】
質問 → 回答

【推論モード】
質問 → 🧠 考える（推論） → 回答
```

推論の段階でAIは次のようなことを行います：

| 推論の内容 | 例 |
|-----------|-----|
| **問題の分解** | 「この問題は3つの部分に分けて考えよう」 |
| **知識の照合** | 「この事例に似たケースを思い出すと…」 |
| **自己修正** | 「あ、最初の考えは間違っていた。こちらの方が正しい」 |
| **複数の選択肢の比較** | 「AとBとCの方法があるが、今回はBが最適だ」 |

### なぜ推論が重要なのか

推論なしのAIが「直感」で答えるとすれば、推論ありのAIは「熟考」してから答えます。特に数学や論理パズル、複雑な問題では推論モードの方が**正解率が大幅に向上**します。

---

## 第2章：推論の仕組みを体験しよう

### やってみよう①：推論をONにしてみよう

入力欄の左側にある **🧠 脳アイコン** ボタンを押すと、推論モードが**ON**（アイコンがアンバー色に光る）になります。

推論ONの状態で、次のプロンプトを試してみましょう。

```
1から100までの整数の合計を求めてください。計算の手順も教えてください。
```

> 💡 AIの回答の上に「思考プロセス」が表示されます。AIが答えを出す前にどのように考えているかを確認しましょう。

---

### やってみよう②：推論OFF/ONで比べてみよう

次に、推論をOFFにして同じ質問をして、回答の違いを比べてみましょう。

推論OFFで試すプロンプト：

```
次の三段論法は正しいですか？「すべての鳥は空を飛べる。ペンギンは鳥だ。よってペンギンは空を飛べる。」
```

続けて推論をONにして同じ質問をしてみましょう。

> 💡 推論モードでは、前提が誤っているかどうかを「考えながら」検証するプロセスが見えます。

## 第3章：推論からAIエージェントへ

### 推論能力がもたらした新しい進化

推論能力が十分に発達したとき、AIは単なる「質問応答マシン」を超えた進化を遂げました。それが **AIエージェント** です。

```
【チャットAI】（これまで）
人間 → 質問 → AI → 回答 → 人間

【AIエージェント】（これから）
人間 → 目標を与える → AI が自律的に考え・行動・修正を繰り返す → 結果
```

### AIエージェントの特徴

| 特徴 | 内容 |
|------|------|
| **計画立案** | 目標達成のためのステップを自ら設計する |
| **ツール利用** | 検索・計算・コード実行などの外部ツールを使う |
| **自己修正** | 実行結果を見て計画を修正しながら進める |
| **長期タスク** | 複数ステップにわたる作業を自律的に完遂する |

### 具体例：AIエージェントができること

**通常のAI（チャット）の場合：**

> 「旅行の計画を立てて」→ テキストでプランを提案するだけ

**AIエージェントの場合：**

1. 🔍 フライトと宿の空き状況を検索する
2. 💰 価格と条件を比較・検討する
3. 📅 カレンダーと照合して最適な日程を選ぶ
4. ✅ 予約まで自動で完了させる

つまり、**「考えるだけ」から「動く」へ**の進化です。

---

### やってみよう③：エージェント的な思考を体験

推論をONにして、多段階のタスクを依頼してみましょう。

```
私は来月、初めて東京から京都へ日帰り旅行を計画しています。予算は交通費含めて2万円以内です。どのように計画を立てるべきか、検討すべき要素を整理しながら考えてください。
```

> 💡 AIが計画を立てる際の「考え方の構造」に注目してください。これが複数のツールと組み合わさると、エージェントとして自律的に動けるようになります。

---

## まとめ

| テーマ | ポイント |
|--------|---------|
| **推論** | 答える前に考えるプロセス。問題分解・自己修正・比較検討を行う |
| **推論の効果** | 複雑な問題や論理的な問いで特に威力を発揮する |
| **AIエージェント** | 推論＋ツール利用＋自己修正＝自律的に目標を達成できるAI |
| **今後の展望** | AIは「聞かれたら答える道具」から「目標を与えると動くパートナー」へ進化している |

AIの推論能力は、単に「賢くなる」だけでなく、AIのあり方そのものを変えつつあります。この進化を理解することで、AIをより効果的に、そして安全に活用できるようになります。
